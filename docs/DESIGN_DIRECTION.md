# Matchday Command — Design Direction

## Core Concept
A living matchday operations map.

## Style
- Modern Google/Material structure
- Apple-level clarity and polish
- Friendly and energetic
- Professional enough for stadium operations
- Cinematic, but still usable

## Main Inspiration
- Large immersive hero sections
- Strong typography
- Interactive map as the main product
- Contextual panels instead of generic dashboard cards
- Smooth transitions and meaningful motion

## Map Direction
- Google Maps-inspired operational experience
- Custom stylized stadium/city map
- Gates, transit, crowd pressure, incidents, and accessibility routes
- Clickable zones with focused information panels

## Colour Direction
- Deep navy base
- Warm off-white surfaces
- Electric blue/cyan for navigation and AI
- Green for safe conditions
- Amber/red only for warnings

## Avoid
- Generic purple AI gradients
- Excessive glassmorphism
- Boring admin dashboard layouts
- Fake FIFA branding
- Random animations
- True 3D complexity that risks the deadline

## Scope Rule
Build one unforgettable interactive stadium-map experience first.
Then redesign the supporting pages around the same visual system.
